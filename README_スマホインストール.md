# スマホに「アプリとして」入れる（PWA化）

このフォルダは **PWA（ホーム画面にインストールできるWebアプリ）** 対応版です。
※重要：PWAは **HTTPS** で開いたときに「インストール」可能になります（例外：PC/Androidの `localhost`）。

---

## 1) 一番かんたん：無料ホスティングに置く（おすすめ）
どれか1つでOKです。

### A. Netlify Drop（最短）
1. Netlify にログイン
2. このフォルダ一式（index.html / sw.js / manifest / icons）をアップロード
3. 発行された **https://〜** のURLをスマホで開く
4. Android：メニュー →「アプリをインストール」  
   iPhone：共有 →「ホーム画面に追加」

### B. GitHub Pages
1. GitHub に新規リポジトリ作成
2. このフォルダ一式をアップロード（root）
3. Settings → Pages → Deploy
4. 表示された **https://〜** をスマホで開く → ホーム画面に追加

---

## 2) PCから同一Wi‑Fiで配る（ローカル運用）
PWAの条件（HTTPS）があるため、**HTTPだけのローカル配布だとインストール/オフラインが動かない**ことがあります。

### 簡易テスト（インストールはできない可能性あり）
- PCでローカルサーバを立てる（例：`python -m http.server 8000`）
- スマホで `http://PCのIP:8000/` を開く  
→ 「動作確認」には使えます。

### 本気でローカルPWAにしたい場合（HTTPS）
- 例：mkcert 等でローカル証明書を作り、HTTPSサーバで配布  
（手順は環境により変わるため、必要ならこちらで端末/OSに合わせて作業手順を書きます）

---

## 3) スマホでのインストール手順
### Android（Chrome/Edge）
- 右上（︙）→「アプリをインストール」または「ホーム画面に追加」

### iPhone / iPad（Safari）
- 共有（□↑）→「ホーム画面に追加」

---

## うまくいかないとき
- URLが **https://** になっているか
- ブラウザが Chrome/Edge/Safari か
- すでにインストール済み（ホーム画面に同じアイコンがある）か
